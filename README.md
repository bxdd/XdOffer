# XdOffer

本项目总结了笔者大学实习和暑期夏令营二十余场面试的经验，旨在指导如何从零开始准备自己的简历、实习和夏令营面试笔试。

## 1 缘由

笔者是同济大学 CS 17级的学生，目前是大四，面试经验不算特别丰富，但是还是在这里总结一下，一是可以对学弟学妹有帮助，二是也避免忘掉前段紧张而充实的日子
